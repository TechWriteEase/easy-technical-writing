专栏合集

芯算故事

不谈玄虚的概念，只讲实在的技术，也聊 AI 的台前幕后。这里是 AI 芯片、大模型与行业趋势的「深度解读站」—— 拆解芯片手册语法，追踪工具迭代升级，科普大模型训练与推理的核心逻辑；同时用轻松视角聊 AI 热点、吐槽行业现象，让高冷黑科技变得接地气。 用通俗的语言，把硬核「芯」算力与 AI 前沿知识，变成听得懂、用得上的技术内容。

Doc As Code

用工程师的方式管理一切。聚焦文档工程化与版本管控，传递「文档与代码同权」的理念。分享 Doc As Code 落地流程、工具选型，以及 Git 从入门到进阶的命令实操、分支管理、协作规范，帮团队搭建高效、可追溯的文档与代码协同体系。

信息设计

好的内容，不止是好用，还要好看。信息设计的本质是降低用户认知成本。专栏聚焦技术内容的视觉表达：从配色方案、字体选择，到图表逻辑、排版布局， 拆解技术文档的视觉优化技巧，让硬核内容既清晰又有专业美感，提升传播效率与阅读体验。

DITA 实战

技术写作领域绕不开的 DITA 工具实战指南。专栏聚焦 Oxygen XML Author，从零基础入门到熟练应用，提供由浅入深的科普与实用指导。 涵盖界面操作、DITA 文档创作、模板定制、批量处理、协同工作等核心场景，帮技术写作者快速掌握这款专业工具，提升文档创作效率与标准化水平。

脚本实战

用技术简化工作，用脚本提升效率。专栏聚焦 Python 脚本实战，覆盖技术写作、办公自动化、数据处理等核心场景， 分享从文档管理、变量替换到批量处理的实用脚本，帮技术写作者把重复工作交给代码，释放时间用于高价值内容创作。

知识管理

你的知识，就是你的职场护城河。专栏聚焦「输入 - 整理 - 输出 - 复用」的完整知识闭环，分享个人知识沉淀、体系搭建、经验复用的方法， 帮你把零散的技术积累转化为可迁移的职场能力。
